# Sesión webpack

Antes de iniciar la sesión tu computadora debe de tener los siguientes programas instalados:

- Node
- NPM
- Git
- Cuaquier editor de su preferencia

# Carpetas

Para ésta sesión, se recomienda tener la siguiente estructura de carpetas:

```
|-- dist
      |-- bundle.js
|-- node_modules
|-- package-lock.json
|-- package.json
|-- src
      |-- index.js
|-- webpack.config.js
```
